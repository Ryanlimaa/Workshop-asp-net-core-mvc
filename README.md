# Sistema Web com ASP .Net Core MVC e Entity Framework
Este é um pequeno sistema de vendedores, seus departamentos e suas vendas desenvolvido no padrão MVC com template engine Razor com layout Bootstrap, ele contém operações CRUD e pesquisas usando LINQ. Este projeto é uma exelente intodução ao desenvolvimento web com ASP.NET Core MVC e o acesso as banco de dados MySQL com Entity Framework.
